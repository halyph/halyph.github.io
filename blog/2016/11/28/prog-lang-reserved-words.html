Reserved Words of Programming languages

28 November 2016

Tags: ruby | python | java | scala | golang | cpp | c |

Amount of reserved words per programming language:

  • C++ - 92
  • C89 - 32
  • C99 - 37
  • C11 - 42
  • Python 2 - 31
  • Python 3 - 33
  • Ruby - 41
  • Golang - 25
  • Java - 53
  • Scala - 50

As You can see Golang has the minimal amount of reserved words.
